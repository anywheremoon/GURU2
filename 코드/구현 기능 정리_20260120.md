
## 구현된 주요 기능 (20260120 수정)

### 1️⃣ 복약 스케줄 관리 (DB 기반)

* 약 정보(`medicine`)
* 복약 일정(`dose_schedule`)
* 일별 복용 기록(`intake_log`)
* SQLite 내부 DB 사용

---

### 2️⃣ 오늘의 복약 체크 화면 (TodayFragment)

* 오늘 날짜 기준 복약 스케줄 자동 조회
* 복용 시간대별(아침 / 점심 / 저녁 / 취침 전) 그룹화
* 체크박스를 통한 **복약 여부 체크**
* 체크 시 `intake_log` 테이블에 자동 저장
* 이미 기록된 복약 여부는 체크 상태 유지

---

### 3️⃣ 복약 세트 RecyclerView (DoseSetAdapter)

* 시간대별 복약 세트 UI 구성
* 아이콘으로 시간대 구분

  * 🌅 아침 / 🌞 점심 / 🌙 저녁 / 🌜 취침 전
* 약 목록을 동적으로 생성
* 체크박스 클릭 시 DB INSERT / UPDATE 처리

---

### 4️⃣ 오늘 복약 요약 표시

* “오늘 먹어야 할 약” 목록 표시
* 복용 완료된 약은 회색 처리
* 미복용 약은 강조 표시

---

### 5️⃣ 달력 기반 복약 기록 조회 (CalendarFragment)

* `MaterialCalendarView` 사용
* 날짜별 복약 상태 시각화

  * 🟢 전부 복용 완료
  * 🟠 일부 복용
  * 🔴 미복용
* 날짜 클릭 시:

  * 해당 날짜의 복약 상세 상태 표시
  * 시간대별 복용 여부(✅ / ❌) 출력


---

## 🛠 사용 기술 스택

* **Language**: Kotlin
* **DB**: SQLite (SQLiteOpenHelper)
* **UI**

  * Fragment 기반 구조
  * RecyclerView
  * MaterialCalendarView
* **외부 라이브러리**

  * `com.prolificinteractive:material-calendarview`
